## Tips Chụp Screenshot

1. **Độ phân giải**: Sử dụng độ phân giải cao (ít nhất 1920x1080)
2. **Định dạng**: PNG cho quality tốt nhất, hoặc JPG để giảm dung lượng
3. **Crop**: Loại bỏ các phần không cần thiết, chỉ giữ nội dung quan trọng
4. **Highlight**: Có thể sử dụng tool để highlight các nút/option quan trọng
5. **File size**: Nén hình nếu file quá lớn (khuyến nghị < 500KB/file)

## Tools Chụp Screenshot

- **Windows**: Snipping Tool, Snip & Sketch (Win + Shift + S)
- **MacOS**: Command + Shift + 4
- **Greenshot**: Free tool with annotation features
- **ShareX**: Advanced screenshot tool cho Windows
